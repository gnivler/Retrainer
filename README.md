#### build instructions (steam and gog window versions)
```
1.  install BattleTech v1.8 or higher (or Battletech and ModTek)
2.  git clone https://github.com/gnivler/Retrainer
3.  to link references, from your installed BattleTech game
    copy directory `BATTLETECH\BattleTech_Data\Managed\`
    into parent-directory of `Retrainer\`
    e.g.:
        C:\development\Retrainer\
        C:\development\Managed\
4.  build this project in Visual Studio (output-file `Retrainer\bin\Debug\Retrainer.dll`)
```
